# go-testcolorize

Colorize your go test output!

[![asciicast](https://asciinema.org/a/ctmwd488pjyiemdzl667yx757.png)](https://asciinema.org/a/ctmwd488pjyiemdzl667yx757)

## Installation

```shell
go get -u github.com/apriendeau/go-testcolorize/cmd/gtc
```

## Example:

```shell
$ go test -v ./... | gtc
```
